UC00604 — Programar para a Web (Frontend)

Desenvolvimento de conteúdos web com HTML e CSS

Objetivos

  • Compreender a estrutura de páginas web utilizando HTML.
  • Programar hiperligações HTML para navegação entre páginas.
  • Aplicar estilos com regras de sintaxe CSS.
  • Construir interfaces simples e responsivas.

Competências Essenciais

  • Domínio de elementos HTML fundamentais como <a>, <div>, <header>, <footer>.
  • Criação de estilos com seletores e propriedades CSS.
  • Desenvolver layouts responsivos para diferentes tamanhos de ecrã.
  • Aplicar boas práticas de estrutura e semântica web.
