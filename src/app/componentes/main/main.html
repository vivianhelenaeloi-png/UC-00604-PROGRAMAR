UC00604 — Programar para a Web (Frontend)

Desenvolvimento de páginas web utilizando HTML e CSS

Objetivos da Unidade

  • Compreender a estrutura e funcionamento de páginas web em HTML.
  • Programar hiperligações para criar navegação entre páginas web.
  • Aplicar estilos visuais utilizando regras de sintaxe CSS.
  • Desenvolver interfaces simples, claras e responsivas.

Competências Desenvolvidas

  • Utilização correta de elementos HTML como a, div, header, footer.
  • Aplicação de seletores e propriedades básicas de CSS.
  • Criação de layouts ajustáveis a diferentes tamanhos de ecrã.
  • Implementação de boas práticas de estruturação semântica.
